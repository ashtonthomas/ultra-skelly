# ultra-skelly

Collaboration repo for integrating AI-driven orchestration with the SVI 6.5-ft animatronic skeleton ("Ultra Skelly") for Halloween 2026.

## Goals

- Integrate a Raspberry Pi, microphone, and camera(s) with the skeleton (speaker, servos, lights)
- Orchestrate an AI-driven interactive experience (Gemini Live + complementary LLM callouts)
- Package the tooling so other Ultra Skelly owners can pick it up and remix it

## Status: brainstorming

This repo is currently a notebook, not a codebase. All planning, ideas, and design notes live in [`docs/`](./docs). Once we converge on direction we'll start scaffolding code.
